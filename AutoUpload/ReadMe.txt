@초기 설정
 사용자 블로그 폴더의 경로를 8행 set path= 옆에 입력
 ex) set path=C:\Users\smart\Desktop\Blog\

 컴퓨터가 실행되는 순간 이 프로그램이 한 번 실행되게 하고 싶으면 
 C:\Users\자신의PC사용자명\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup 이 경로에 AutoUploader.bat 파일을 복사해서 붙여넣는다.
 참고로 이미 위의 경로 변경이 된 파일로 해야한다.
 
=================================================================================================================

@사용방법
 1. 사용자가 만든 TIL 파일을 yyyy.mm.dd-TIL.md 제목 형식에 맞춰서 Blog\New 폴더에 집어 넣는다.
 2. 직접 업로드 하고 싶은 경우 AutoUploader.bat을 직접 실행시킨다.
 3. 프로그램에서 나오는 문구에 따라 행동한다.
 4. cmd에서 이 프로그램을 실행시키고 싶다면, cmd를 실행시킨 뒤 AutoUploader.bat 파일을 드래그해서 cmd에 넣고 enter 누른다.

=================================================================================================================

@프로그램 구현 원리
 1. 오늘 날짜에 해당한는 yyyy.mm.dd-TIL.md 파일이 Blog\New 폴더에 존재하는지 확인한다.
 2. 없으면 패스, 있으면 git 프로그램을 통해 업로드 하는 일련의 과정을 거친다.
 3. git 프로그램으로 업로드가 끝나고 나면 move 명령어를 통해서 Blog\New 폴더에 위치한 yyyy.mm.dd-TIL.md 파일을 Blog\Uploaded 폴더로 이동시켜 안전하게 보관한다.
